# MultiSales Logo

The MultiSales logo features an elegant corporate design with:

- Company name "MULTISALES" in golden/cream typography
- Subtitle "MAROC" beneath the main text
- Geometric diamond-pattern symbol above the text
- Deep navy blue background
- Premium, professional appearance suitable for business applications

## Files

- `multisales_logo.jpg` - Main company logo (navy blue background with golden text)

## Usage

Use the logo in your Flutter app like this:

```dart
Image.asset('assets/images/multisales_logo.jpg')
```

## Brand Guidelines

- Maintain aspect ratio when resizing
- Logo works well on light backgrounds due to dark navy background
- Minimum size: 24x24 pixels for icons, 120px width for logos
- The golden text provides excellent contrast against the navy background
